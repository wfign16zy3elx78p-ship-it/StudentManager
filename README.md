\# Ứng dụng Quản lý Sinh viên (Student Manager)



Môn học: Introduction to Software Engineering – CMU SE100 2025  

Nhóm 2:Huỳnh Huy Cường ,Hồ Ngọc Hải,Vo Viết Hòa



\##  Mô tả

Ứng dụng quản lý thông tin sinh viên bằng C# Windows Forms, hỗ trợ:

\- Thêm/Sửa/Xóa/Tìm kiếm

\- Lưu/mở file CSV

\- Thống kê, sắp xếp, tính điểm TB

\- Giao diện Material Design



\## Công nghệ

\- C# (.NET Framework 4.8)

\- Windows Forms

\- MaterialSkin.2 (UI)



\##  Cấu trúc

\- `src/` – Mã nguồn

\- `docs/` – Tài liệu SRS, Test Plan, User Manual...

\- `slides/` – Slide thuyết trình



\##  Cách chạy

1\. Mở `StudentManager.sln` bằng Visual Studio 2022

2\. Build và chạy (F5)
